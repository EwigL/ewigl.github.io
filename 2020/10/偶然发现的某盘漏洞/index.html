偶然发现的某盘"漏洞"

发表于 2020-10-08 更新于 2020-10-21 分类于 笔记

  • 偶然得到一个网盘链接但是没有密码

  • 习惯性的 f12，一眼就看到一个 fileview…这是明示

  • 展开，发现了一个 id 为 folder 的元素,而且还有两个链接

  • 点击链接即可访问子文件夹

  • 限制：只可以访问子文件夹以及子文件夹中的文件,无法访问分享根目录文件。
