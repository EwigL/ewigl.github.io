Java Scanner中文输入乱码问题

发表于 2021-01-10 分类于 笔记

  最近在做Java项目时遇到了一个问题：在使用JDBC向MySQL数据库中插入数据时中文值会变成乱码，但是从数据库中查询数据时中文显示正常。因此乱码产生原因应该在输入过程。

  推测原因是使用Scanner类创建的输入流输入中文时编码转换产生错误。

示例结果如下

  可以看到输入的中文变成了乱码，但是预定义的字符串可以正常输出，因此确定是输入时编码存在问题。
  键盘的输入流的默认格式是GBK格式，控制台的字符编码以及文件编码均为UTF-8，因此造成了乱码。
  解决：更改代码Scanner sc = new Scanner(System.in);在实例化Scanner对象时指定编码类型为GBK，即Scanner sc = new Scanner(System.in, "GBK");。
