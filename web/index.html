mouseover 与 mouseenter 的区别

  mouseenter 不会冒泡。它不会被它本身的子元素的状态影响到。
  mouseover 会被它的子元素影响到,在触发子元素的时候,mouseover 会冒泡触发它的父元素。

文档碎片操作：减少重排，提高效率

  将需要添加的大量元素先添加到文档碎片中，再将文档碎片添加到需要插入的位置，大大减少 DOM 操作，提高性能。

call( );apply( );bind: this 重定向函数

  • 第一个参数均为 this 指向对象
  • call()与 apply()均立即调用，bind 返回的是一个新的函数
  • call()原函数参数后移一位，apply()原函数参数作为数组传入，bind()除返回值是函数以外，参数与 call()一样
